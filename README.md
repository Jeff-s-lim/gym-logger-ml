# gym-logger-ml
ongoing gym logger project with linear regression model to predict future weights and reps.
gym logger using Hevy's built in log export feature.

## current
- initial CSV loading and cleaning logic implemented
- support for multiple hevy files
- support for different weight metrics
- created a summarized weekly data for model training
- built initial regression model
- mae vs model mae

## planned future implementations
- weight input system per week with correlation
- prediction weight system with "cutting" & "bulking" features
- proper frontend using react/typescript
- user authentication
- database using postgresql
- chatbot feature using small llm
- flask rest api for frontend/backend communication
- user specific progression charts
- per exercise and per user model evaluation
- full import history showing uploaded files
